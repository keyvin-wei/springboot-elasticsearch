# springboot-elasticsearch

springboot集成elasticsearch  
1、ES连接客户端使用High Level REST Client（官方推荐）  
2、集成swagger接口  
3、集成thymeleaf编写页面  
4、需要事先安装elasticsearch  

## 版本

组件框架 | 版本
--- | ----
springboot | 2.1.7.RELEASE
swagger-ui | 2.9.2
elasticsearch-rest-high-level-client | 6.6.0
elasticsearch | 6.6.0
